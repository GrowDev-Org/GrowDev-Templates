# GrowDev — Templates

Este repositório contém os **templates e padrões oficiais** utilizados nos projetos do GrowDev.

O objetivo é:
- Padronizar issues e pull requests
- Facilitar a organização dos projetos
- Incentivar boas práticas desde o início

---

## Estrutura inicial

```
.github
├── ISSUE_TEMPLATE/
│ ├── bug.md
│ ├── feature.md
│ └── task.md
└── pull_request_template.md
README.md
```

## Como usar

Todos os projetos do GrowDev devem:
- Utilizar estes templates
- Criar issues bem descritas
- Abrir pull requests claros e objetivos

Padronização é parte do aprendizado.
